== Description ==

Easily create WordPress WooCommerce login popup on your store to have a professionally looked beautiful ways to let your customers in.

<h3>WooCommerce Modal Popup Features</h3>
Turn your visitors into valuable customers with elegance. Using this plugin, you'll have the following features on your popup:

* <strong>Login Area</strong>. Show store login at ease without having your site reload and navigate to new page. Using this WooCommerce login modal box you eliminate the waiting time of your customers to login into your shop.
* <strong>Registration Area</strong>. Welcome your brand new customers better! Let them register fast using the registration popup option.
* <strong>Reset Password Area</strong>. Your beloved customers might got system generated password or simply forgot what they've input on your store which sometimes create difficulties while shopping. Don't add any more hindrances and let them regenerate there WooCommerce password at ease via forgot password popup feature.
* <strong>Customer Dashboard</strong>. Let your logged-in users perfectly navigate your site. When successfully logged-in, the modal box content will change to WooCommerce dashboard navigation links.

<h3>WooCommerce Login and Registration Shortcode </h3>
Don't like the default WooCommerce login and registration design? Use the plugin's shortcode feature to have a brand new page which is similar to your popup branding. Less confusion more shopping!

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. Go to WooCommerce > Login Popup and upload background image; and enable the PopUp option
4. Then go to Appearance > Menu and add 'woo-login-popup-sc-open' class to the link you want the popup to open when clicked.
